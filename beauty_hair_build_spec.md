> SaaS 랜딩페이지 양산 가이드의 체크리스트 10개 항목을 채운 버전

| # | 항목 | 내용 |
|---|------|------|
| 1 | 타깃 업종 | 미용실 (헤어살롱, 1인샵, 프랜차이즈) |
| 2 | 헤드라인 | "헤어 모델 촬영, 이제 AI로 10초면 끝." |
| 3 | 페인포인트 | 아래 상세 |
| 4 | 핵심 기능 3가지 | 아래 상세 |
| 5 | 데모 영상/GIF | 시술 사진 → AI 헤어 모델 변환 과정 |

| 7 | 가격표 | 아래 상세 |
| 8 | 기능 체험 UI | 시술 사진 업로드 → 헤어 스타일/모델/배경 선택 → 생성 |
| 9 | 카카오톡 채널 | 뷰티 전용 채널 개설 |
| 10 | 도메인/브랜드명 | 대표 확정 (예: HairShot AI, ModelHair 등) |

## 타깃 분석

### 누구인가

- 미용실 원장님 (1인샵~중소 살롱)
- 헤어 디자이너 (프리랜서 포함)
- 뷰티 프랜차이즈 마케팅 담당자

### 이들의 핵심 행동

- 인스타그램/네이버 플레이스에 시술 사진을 올려서 고객 유치
- "시술 사진이 많은 미용실일수록 매출이 높다"는 걸 알고 있음
- 헤어 모델을 카페(모델나라 등)에서 구인하거나, 고객에게 촬영 동의를 요청

### 이들이 모이는 곳 (마케팅 채널)

- 네이버 카페: 모델나라, 헤어 디자이너 커뮤니티
- 인스타그램: #헤어디자이너 #미용실원장 #헤어모델구함
- 네이버 블로그: 미용실 운영 노하우
- 카카오톡 오픈채팅: 미용실 원장 모임

## 페인포인트 4가지

| # | 문제 | 상세 |
|---|------|------|
| 1 | **모델 구하기가 너무 어렵다** | 모집글 올려도 적합한 모델이 안 옴. 모질이 안 맞거나, 시간이 안 맞거나, 노쇼 발생 |
| 2 | **초상권 문제** | 고객 시술 후 촬영 요청하면 대부분 "얼굴 모자이크 해주세요". 모자이크 처리된 사진은 헤어스타일 매력이 반감됨 |
| 3 | **비용이 과하다** | 전문 모델 섭외 + 촬영 + 보정 = 1회 100만원 이상. 1인샵/소규모 살롱에겐 부담 |
| 4 | **시간을 뺏긴다** | 모델 구인 → 스케줄 조율 → 시술 → 촬영 → 보정까지 최소 반나절~하루. 그 시간에 고객을 받아야 함 |

## 핵심 기능 3가지

| 기능 | 설명 | 셀러 언어 |
|------|------|----------|
| **AI 헤어 모델 생성** | 시술 사진(뒷모습/옆모습)을 업로드하면 AI가 얼굴이 보이는 정면 모델컷을 생성. 초상권 걱정 제로 | "모델 없이 인스타용 사진 완성" |
| **헤어 스타일 시뮬레이션** | 원하는 헤어 스타일을 AI 모델에 적용해서 미리 보여줌. 고객 상담 시 활용 가능 | "이 스타일 하면 이렇게 됩니다" |
| **SNS 콘텐츠 자동 생성** | 생성된 이미지를 인스타 피드/릴스/네이버 플레이스 최적화 사이즈로 자동 변환 | "바로 올릴 수 있는 콘텐츠" |

## 클로드코드 프롬프트

```
첨부한 문서는 뷰티/헤어 AI 모델 SaaS의 웹사이트 빌드 설계서입니다.
미용실 원장님을 타깃으로 한 AI 헤어 모델 생성 구독 서비스입니다.
이 문서를 기반으로 Next.js + Tailwind CSS 프로젝트를 만들어주세요.

중요 사항:
1. 비주얼 퍼스트 원칙: 메인 페이지 섹션 1~3은 이미지 90% + 텍스트 10%입니다. 모바일 유저가 3초 안에 이탈하므로 글보다 결과물 이미지를 먼저 보여줘야 합니다.
2. 컬러는 로즈 핑크 (#E11D48) 계열로, 따뜻하고 부드러운 뷰티 톤입니다.
3. 타깃이 미용실 원장님(40~50대 포함)이므로 UI가 최대한 심플해야 합니다. 복잡한 인터랙션 없이 "사진 올리고 버튼 누르면 끝"이어야 합니다.
4. 모바일 반응형 필수. SNS(인스타그램)에서 유입되는 모바일 유저가 대부분입니다.

우선순위:
- 1순위: 메인 랜딩페이지 (8섹션)
- 2순위: 기능 체험 페이지 - AI 헤어 모델 생성 탭
- 3순위: 스타일 시뮬레이션, SNS 변환 탭
- 4순위: 결제 페이지

AI API는 아직 연동하지 말고, 목업 데이터로 UI를 먼저 완성해주세요.
```
