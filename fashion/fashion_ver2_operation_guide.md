# Ver.2: 프리미엄 구축 대행 — 운영 가이드

> Ver.2는 별도 개발이 아닌, Ver.3 플랫폼을 활용한 영업 채널입니다.
> 이 문서는 "빌드 설계서"가 아니라 "운영 가이드"입니다.

---

## 한 줄 정의

"Ver.3 플랫폼으로 만들어주고 고단가 수주" — 추가 개발 없이 영업만

## 타깃

중형 패션 브랜드, 자사몰 리뉴얼 원하는 기업, 신규 런칭 브랜드

---

## 추가 개발 사항

```
없음.

Ver.3 마케팅 랜딩 하단에 "엔터프라이즈" 섹션을 추가하는 것이 전부.
이 섹션은 Ver.3 빌드 시 이미 포함되어 있음.
```

---

## 엔터프라이즈 섹션 (Ver.3 랜딩 하단에 포함)

```
─────────────────────────────────────
"브랜드 맞춤 구축이 필요하신가요?"

대규모 브랜드를 위한 프리미엄 구축 서비스
전담 매니저가 브랜드에 맞는 AI 쇼핑몰을 구축해드립니다.

포함 사항:
✅ 브랜드 맞춤 디자인 (전용 테마)
✅ AI 모델 커스텀 학습 (브랜드 전용 모델)
✅ 기존 쇼핑몰 데이터 마이그레이션
✅ 전담 매니저 배정
✅ 런칭 후 3개월 운영 지원

[문의하기 →] (카카오톡 채널)
─────────────────────────────────────
```

---

## 워크플로우

```
1. 브랜드가 Ver.3 랜딩 엔터프라이즈 섹션에서 문의
2. 카카오톡/미팅 → 요구사항 파악
3. Ver.3 플랫폼에서 쇼핑몰 생성 (우리가 셀러 계정으로 직접)
4. 브랜드 맞춤 커스터마이징
   - 전용 테마 디자인
   - 로고/배너 적용
   - AI 모델 커스텀 학습 (브랜드 톤에 맞는 모델)
   - 기존 상품 데이터 마이그레이션
5. 브랜드에 관리자 계정 이관 + 운영 교육
6. 런칭
7. 3개월 운영 지원 → 이후 월간 유지보수 (엔터프라이즈 플랜)
```

---

## 가격 모델

| 항목 | 가격 |
|------|------|
| 초기 구축비 | 500~2,000만원 (브랜드 규모에 따라) |
| 월 유지보수 | Ver.3 엔터프라이즈 플랜 (399,000원/월) |
| AI 모델 커스텀 학습 | 별도 견적 (100~500만원) |
| 데이터 마이그레이션 | 별도 견적 (상품 수에 따라) |

---

## 영업 채널

| 채널 | 방법 |
|------|------|
| 인바운드 | Ver.3 랜딩 엔터프라이즈 섹션 → 카카오톡 문의 |
| 아웃바운드 | Ver.1 유료 구독자 중 성장한 셀러에게 업셀 제안 |
| 포트폴리오 | 구축 완료된 쇼핑몰을 Ver.3 랜딩 데모에 추가 |

